## Bloque 9 — Fix INP (parcial) ✅

Cambios aplicados a `layout/theme.liquid`:
- `jquery.js`: añadido `defer="defer"` (estaba síncrono — bloqueante en render)
- Añadidos `dns-prefetch` para: googletagmanager, google-analytics, klaviyo, smile.io, judge.me, pleiful
- Añadido `preconnect` para googletagmanager

**Limitación honesta:** Pleiful, Smile.io, Klaviyo, Bing UET, fb pixel **no se cargan desde theme.liquid** — se inyectan vía App Embeds (settings_data.json) o Online Store 2.0 app blocks. Para reducir INP por debajo de 200ms hay que:
1. Desactivar App Embeds no críticos en Customizer (Online Store → Theme → Customize → App embeds)
2. O usar Shopify's "Hydrogen" / web pixel sandbox para chats
3. Bing UET, fb pixel: revisar en Customizer → App embeds

He hecho el máximo que se puede hacer en el theme code. El resto requiere decisión humana en el Customizer.
